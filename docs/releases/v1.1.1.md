## MacFanControl v1.1.1

本版本集中修复 Developer ID 分发后的风扇控制服务连接、M4 SMC 数据兼容和旧版 Helper 迁移问题。

### 风扇控制与 M4 兼容

- 修正 AppleSMC 数据结构的 80 字节 C ABI 布局。
- 支持 M4 风扇键使用的 `flt ` 浮点格式，并保留旧机型的 `fpe2` 格式。
- 保留 SMC 键元数据，避免读取阶段丢失数据类型和长度。
- Helper 重连期间不再错误降级为直接 SMC 调用。

### Helper 服务稳定性

- 使用 SMAppService 和双向代码签名校验保护 XPC 风扇控制。
- 修复已启用服务的连接重试和状态提示。
- 通过 launchd 服务标签可靠清理旧版 Helper、plist、可执行文件和 socket。

### 安装

1. 下载并解压 `MacFanControl_v1.1.1.zip`。
2. 将 `MacFanControl.app` 移入 `/Applications`。
3. 首次运行时根据提示启用风扇控制服务，并在系统设置中批准后台项目。

要求 macOS 13.0 或更高版本。本发布包使用 Developer ID 签名、通过 Apple 公证并装订公证票据。
