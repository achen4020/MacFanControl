## MacFanControl v1.1.0

本版本在风扇监控与控制基础上新增完整的区域截图工作流，并补齐启动磁盘、网络状态、配置持久化和应用启动行为。

### 新增区域截图与编辑

- 默认通过 `Control + Shift + A` 触发区域截图，可在设置中修改或恢复快捷键。
- 支持在鼠标所在显示器自由框选，提供尺寸提示、放大镜、反向拖动、`Esc` 和右键取消。
- 编辑器支持裁剪、矩形、箭头、画笔、文字、马赛克、选择、移动、缩放、删除、撤销和重做。
- 支持 PNG/JPEG 保存、复制到剪贴板，以及通过 `Command + V` 打开剪贴板图片。
- 支持同时打开多个编辑窗口和未保存内容关闭确认。

### 系统监控增强

- 显示当前 macOS 启动磁盘的已用容量、总容量和使用比例。
- 显示当前活跃物理网络接口合计的实时上传与下载速度。
- SSD 温度仅使用能够明确识别的 NAND 传感器；无法可靠对应的传感器不再展示。

### 风扇控制与稳定性

- 自动控制配置和自定义曲线在应用重启后继续保留。
- 应用启动后立即开始风扇监控和自动控制，无需先点击菜单栏图标。
- 修复网络计数器回绕和高位计数器重置导致的异常速率。
- 修复截图编辑器初始窗口被 SwiftUI 压缩、图片不可见的问题。
- 使用稳定指定要求进行开发签名，避免重新构建后屏幕录制权限因 `cdhash` 改变而失效。

### 安装与权限

1. 下载并解压 `MacFanControl_v1.1.0.zip`。
2. 将 `MacFanControl.app` 移入 `/Applications` 后启动。
3. 根据应用提示输入管理员密码安装风扇控制 Helper。
4. 首次截图时在“系统设置 > 隐私与安全性 > 屏幕与系统音频录制”中授权，并重新启动应用。

要求 macOS 13.0 或更高版本。当前发布包使用本地临时签名，未经过 Apple Developer ID 公证。
